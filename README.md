# Azure Enterprise Landing Zone

## Overview

This project provisions an enterprise-style Azure Landing Zone using Terraform.

## Technologies

- Terraform
- Microsoft Azure
- Azure Resource Manager (ARM)

## Current Features

- Resource Groups
- Modular Terraform Design

## Planned Features

- Virtual Network
- Subnets
- Network Security Groups
- Route Tables
- Azure Bastion
- Windows Server
- Ubuntu Server
- Storage Account
- Key Vault
- Log Analytics
- Recovery Services Vault
